Questions fréquentes (FAQ)

Compte et Connexion

Q : J'ai oublié mon mot de passe, comment puis-je le réinitialiser ?

R : Sur l'écran de connexion, cliquez sur "Mot de passe oublié" et suivez les instructions envoyées par email.

Q : Comment supprimer mon compte ?

R : Allez dans Paramètres > Profil > Supprimer mon compte. Attention, cette action est irréversible.

Q : Puis-je changer mon adresse email ?

R : Contactez-nous à contact@padelbuzz.app pour modifier votre adresse email.

Enregistrement de Matchs

Q : Pourquoi mes vidéos ne s'enregistrent pas ?

R : Vérifiez que :

  • Vous avez autorisé l'accès à la caméra dans les paramètres de votre téléphone
  • Vous avez suffisamment d'espace de stockage
  • Votre connexion internet est stable pour l'upload

Q : Combien de temps puis-je enregistrer ?

R : Les matchs peuvent durer jusqu'à 3 heures. Pour des durées plus longues, contactez le support.

Q : Où sont stockées mes vidéos ?

R : Vos vidéos sont stockées de manière sécurisée sur nos serveurs cloud. Vous seul avez accès à vos vidéos privées.

Problèmes Techniques

Q : L'application plante ou ne répond plus

R : Essayez de :

  1. Fermer complètement l'app et la relancer
  2. Redémarrer votre téléphone
  3. Vérifier que vous avez la dernière version de l'app
  4. Réinstaller l'application (vos données sont sauvegardées dans le cloud)

Q : La qualité vidéo est mauvaise

R : Allez dans Paramètres > Qualité vidéo et sélectionnez une qualité supérieure. Notez que cela consommera plus de données et d'espace de stockage.

Q : Les notifications ne fonctionnent pas

R : Vérifiez que les notifications sont activées dans :

  • Les paramètres de l'app (Paramètres > Notifications)
  • Les paramètres de votre téléphone (Réglages > Notifications > PadelBuzz)

Confidentialité et Sécurité

Q : Mes données sont-elles sécurisées ?

R : Oui, nous utilisons un chiffrement en transit et suivons les meilleures pratiques de sécurité. Consultez notre Politique de Confidentialité pour plus de détails.

Q : Puis-je télécharger toutes mes données ?

R : Oui, contactez-nous à contact@padelbuzz.app pour recevoir une copie de toutes vos données personnelles.

Q : Qui peut voir mes vidéos ?

R : Par défaut, seul vous pouvez voir vos vidéos. Vous choisissez explicitement si vous voulez partager un highlight.

Signaler un Bug

Si vous rencontrez un bug ou un problème technique :

  1. Décrivez le problème : Expliquez ce qui ne fonctionne pas
  2. Étapes pour reproduire : Comment avez-vous rencontré ce problème ?
  3. Appareil et version : Modèle de téléphone, version d'iOS
  4. Captures d'écran : Si possible, joignez des captures d'écran

Envoyez ces informations à : contact@padelbuzz.app
